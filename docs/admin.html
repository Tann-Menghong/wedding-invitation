🔐 Admin Login

This static site has no server, so the admin panel talks to GitHub directly using a Personal Access Token. Create a fine-grained token scoped to only the wedding-invitation repo with Contents: Read and write permission, then paste it below. The token is kept only in this browser tab's session storage and sent directly to api.github.com — never anywhere else.

Prefer not to use a token? Edit content.json directly on GitHub.
